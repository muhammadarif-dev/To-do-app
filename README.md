# To-do-app
•	Developed a full-featured task management web application with sidebar navigation layout.
•	Implemented complete CRUD functionality: add, edit, delete, and mark tasks as complete.
•	Features include priority levels (High/Medium/Low), category filters (Work/Study/Personal/General), search, and progress tracking with a circular progress ring.
•	Data persists using localStorage — tasks remain saved after browser is closed.
•	Fully responsive design with mobile hamburger menu and slide-in sidebar.
